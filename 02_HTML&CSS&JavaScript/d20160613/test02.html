HTML이란 HyperText Markup Language의 약자. 여러 태그들로 구성되어 있으며, 각 태그들을 사용하여 원하는 형태의 문서를 만들어 갈 수 있다.
또한 각 태그는 단독 태그를 제외하고는 항상 쌍으로 이루어져 있다.
단독 태그의 경우 끝에 슬래시(/)를 넣어주는 것이 정석.
